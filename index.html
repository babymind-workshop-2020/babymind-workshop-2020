BabyMind:
How Babies Learn and How Machines Can Imitate

Vancouver Convention Center, NIPS 2020
Half day (AM or PM) on Dec xx, 2020, Vancouver, Canada

BabyMind Workshop 2020

Deep neural network models have shown remarkable performance in the tasks such as visual object recognition, speech recognition and autonomous robot control. It has been continuously developed throughout the years that it has even come to the point where the performance of these models surpassed actual human performance in some tasks. However, the use of deep learning techniques is not without its limitations. Since the performance of deep learning heavily relies on the quantity of the data it requires a huge amount of time and effort in order to collect and label information.

Recently, in order to overcome these weaknesses and limitations, deep learning models such as semi-supervised learning, incremental learning, and meta-learning methods have been studied. However, to apply the deep learning model to real-world problems, it still requires a lot of data, and there is no remarkable achievement, especially in terms of the ability to abstract concepts and generalize problems.

In contrast, babies gradually make sense of the environment through their experiences, a process known as learning by doing, without a large amount of labeled data. They actively engage themselves in exploring the world via their own interactions. They gradually acquire the abstract concept of objects, and develop the ability to generalize problems. In this point of view, if we understand how a baby mind develops through learning by doing, we can build machines to imitate. In this workshop, we explore learning by doing situations that can help us to build baby-like intelligence in machines.

In this workshop, we would try to answer the following questions on babies and machines:

  • How does a baby learn by own interactions and experiences?
  • What sort of insights we can acquire from baby mind?
  • How those insights can help us to build smart machines as babies?
  • How machines can learn from babies to do better?
  • How far is the recent machine intelligence from babies?

Call for Papers

In this interdisciplinary workshop, we bring together eminent researchers in Computer Science, Cognitive Science, Psychology, Brain Science and various other related fields to discuss the above questions. We will also invite researchers to submit peer-reviewed papers on the below mentioned topics. We will accept around 15 submissions for poster presentations. Our half-day workshop will have an oral session, a poster session, and an open panel discussion to exchange ideas. Topics of interest include but not limited to:

  • Neuro-cognitive learning system
  • Incremental / continual / life-long / never-ending learning
  • Curiosity-driven self-supervised learning
  • Active learning based on perception-cognition-action cycle
  • Embodied concept learning from real-world interactions
  • Object-centric representation and concept learning
  • Cross-modal / multi-modal concept learning
  • Developmental / cognitive robotics
  • Social and emotional development system
  • Language development system

Submission Instructions

We invite submissions of papers as extended abstract within 4 pages, excluding references or supplementary materials. All submissions must be in pdf format as a single file (incl. supplementary materials) using below templates and submitted through this CMT link. The review process is single-round and double-blind. All submissions have to be anonymized.

  • LaTeX/Word Templates (tar): NeurIPS 2020 LaTeX style file

All accepted papers will be presented as posters during the workshop and listed on the website. Additionally, a small number of accepted papers will be selected to be presented as contributed talks.

Dual Submissions

Note that this workshop will not publish official proceedings. The accepted submission will not be counted as a publication. We encourage submissions of relevant work that has been previously published, or is to be presented at the main conference.
